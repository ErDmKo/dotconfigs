# Neovim configs

## Simple copy comand
```bash
git archive --format=zip --output nvim-config.zip HEAD
```

## Install typescript-language-server
```bash
nvm use 16
npm install -g typescript typescript-language-server
```

## Plugins install
``` 
:PackerInstall
:TSUpdate
```
